# Wordpress コンテナ

## 概要

シンプルな MySQL 利用の Wordpress を構成するコンテナ群です。

## 使い方

```
docker compose up -d
```

## 備考

- 初期状態では http://localhost:8080 でアクセスします。
- MySQL は外部からアクセスできません。
